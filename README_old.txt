# Scheculing App 
- Author: Diego Torres
- email: dtor172@wgu.edu
- Date: 2022-04-26

##Purpose: This purpose of this application is to add, modify, and delete appointments for 
each customer. This includes various scheduling data that is checked to ensure the data 
is valid, times do not overlap, and are with business hours. Customers can also be added, 
modified, and deleted. Reports can be generated based on the collected data.

##Development Enviroment:
- Developed in IntelliJ Community 2021.1.3
- Java SE 17.0.1
- JavaFX-SDK-17.0.1
- mysql-connector-java-8.0.26

##How to Run:
- Open project in IntelliJ and update the path for JavaFX SDK, JDK and JDBC.
- Update the DBConnection class with your DB informaiton. 
- Press Run.

##Additional Report
The additional report provides a count for the number of appointments that are currently
scheduled at each location.
